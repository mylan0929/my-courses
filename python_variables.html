🐍 Python 变量完全指南

什么是变量？

变量是编程中用来存储数据的容器。你可以把变量想象成一个贴有标签的盒子，标签就是变量名，盒子里装的内容就是变量的值。

简单示例

# 创建一个名为age的变量，存储数字25
age = 25

# 创建一个名为name的变量，存储字符串"小明"
name = "小明"

# 打印变量
print("姓名:", name)
print("年龄:", age)

变量命名规则

在Python中，变量名需要遵循以下规则：

  • 只能包含字母、数字和下划线（A-z, 0-9, _）
  • 不能以数字开头
  • 不能是Python的关键字（如if, else, for等）
  • 区分大小写（Age和age是不同的变量）

合法与非法变量名示例

# 合法的变量名
my_variable = 10
variable2 = 20
_user_name = "小红"
MAX_SIZE = 100

# 非法的变量名
2variable = 30  # 不能以数字开头
my-variable = 40  # 不能包含连字符
for = 50  # 不能使用关键字

📌 命名建议：使用有意义的变量名，避免使用单个字母（除非是简单的循环计数器）。Python通常使用小写字母和下划线（snake_case）作为变量命名风格。

变量类型

Python是动态类型语言，不需要显式声明变量类型。Python会自动根据赋值确定变量类型。

基本数据类型

# 整数 (int)
age = 12

# 浮点数 (float)
height = 1.75

# 字符串 (str)
name = "李雷"

# 布尔值 (bool)
is_student = True

# 列表 (list)
fruits = ["苹果", "香蕉", "橙子"]

# 元组 (tuple)
dimensions = (1920, 1080)

# 字典 (dict)
student = {"name": "韩梅梅", "age": 13}

变量赋值

Python中有多种赋值方式：

1. 简单赋值

x = 10
message = "你好，世界！"

2. 多重赋值

# 同时给多个变量赋相同的值
a = b = c = 100

# 同时给多个变量赋不同的值
name, age, height = "王芳", 14, 1.65

3. 交换变量值

x = 5
y = 10
x, y = y, x  # 交换x和y的值

变量作用域

变量的作用域决定了在哪里可以访问这个变量：

1. 全局变量

school = "阳光小学"  # 全局变量

def print_school():
    print("学校:", school)  # 可以访问全局变量

print_school()

2. 局部变量

def calculate():
    result = 10 + 5  # 局部变量
    print("结果:", result)

calculate()
# print(result)  # 这里会报错，因为result是局部变量

3. 使用global关键字

count = 0

def increment():
    global count  # 声明使用全局变量
    count += 1

increment()
print("计数:", count)  # 输出: 计数: 1

⚠️ 注意：尽量避免过多使用全局变量，这会使代码难以维护和理解。

常量

Python没有真正的常量，但约定用全大写字母表示常量（不应该被修改的值）。

PI = 3.14159
MAX_USERS = 100

# 虽然技术上可以修改，但不应该这样做
# PI = 3.14  # 不推荐

类型转换

有时需要在不同类型之间转换：

类型转换示例

# 字符串转整数
age = "12"
age_int = int(age)

# 整数转字符串
student_id = 12345
id_str = str(student_id)

# 浮点数转整数
price = 9.99
price_int = int(price)  # 会截断小数部分

# 布尔值转换
has_homework = "True"  # 注意这是字符串
is_true = bool(has_homework)  # 非空字符串会转为True

高级主题

1. 可变与不可变类型

Python中有些数据类型是可变的（可以修改），有些是不可变的（不能修改）：

# 不可变类型（修改会创建新对象）
x = 10
y = x
x = 20  # y仍然是10

# 可变类型（修改会影响所有引用）
list1 = [1, 2, 3]
list2 = list1
list1.append(4)  # list2也会变成[1, 2, 3, 4]

2. 变量内存管理

Python使用引用计数和垃圾回收机制管理内存：

import sys

a = [1, 2, 3]
print(sys.getrefcount(a))  # 查看引用计数

b = a
print(sys.getrefcount(a))  # 引用计数增加

del b  # 删除一个引用
print(sys.getrefcount(a))  # 引用计数减少

3. 特殊变量名

Python中有一些特殊命名约定：

# 单下划线开头：表示内部使用
_internal_var = "仅供内部使用"

# 双下划线开头：名称修饰（Name Mangling）
__private_var = "私有变量"

# 双下划线开头和结尾：特殊方法
__init__ = "构造函数"

📌 小技巧：使用type()函数可以查看变量的类型：print(type(age))
